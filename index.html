Wind Energy EDW

Wind Energy EDW Design Documents and SQL

Download this project as a .zip file Download this project as a tar.gz file

This database project illustrates the Kimball approach to designing an enterprise data warehouse for use by business analysts focused on electrical energy generated by wind turbines.

Documents included with the project:

1) An executive summary of the design process and resulting dimensional model.

2) An Excel spreadsheet documenting the DDL of the fact and dimension tables.

3) An image of the ERD model of the deployed database.

4) And of course the SQL code (for Microsoft SQL Server 2014).
